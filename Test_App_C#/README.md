# Test App C# Suite

Collection of C# WinForms utilities used to exercise firmware interfaces and visualize data.

## Projects
- `Artemis_Esp32_Test_App/Artemis_Esp32_Test_App.sln` – WinForms client targeting .NET Framework 4.8 for interacting with Artemis/ESP32 hardware.
- `Farand_Chart_Lib_Ver3/` – Charting library and demos packaged as a Visual Studio solution.
- `Simple_Client_LAN_Control_Revised/` – LAN control test client with its own solution file.

## Building and running
1. Install Visual Studio 2019 or later with the **.NET Framework 4.8** workload.
2. Open the desired `.sln` file from its subfolder.
3. Restore NuGet packages when prompted, then build and run the WinForms application in Debug or Release mode.
